# 🔐 Coding Contest Login Page

## 📌 Overview

This project is a **Login Page for a Coding Contest Platform**, developed during our college creathon. It provides a clean and user-friendly interface for participants to log in and access the contest system.

The project focuses on **frontend design, user experience, and basic authentication workflow**.

---

## 🎯 Objective

* To design a responsive and modern login interface
* To simulate a real-world coding contest login system
* To enhance teamwork and web development skills

---

## 🚀 Features

* 🔑 Login form with email
* ✅ Basic input validation
* 🎨 Attractive and responsive UI
* ⚡ Lightweight and fast loading

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Tools Used:** VS Code, GitHub
